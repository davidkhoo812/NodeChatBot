To add a key, please fill out the following fileds and click the submit button

Please note that only administrators have access to viewing the full list of currently available keys. Users can only view and delete their own keys.

If you wish to keep the game title a secret from other users, please enter 'Mystery Game' into the title field and enter the title into the 'Secret Title' field instead; this way we can track what games were given away, and you can still give out the game as a mystery.

You may enter any combination of obfuscated and non-obfuscated key; if you enter both, the un-obfuscated key will be displayed in private, and the obfuscated key will be used publicly. If your item is a tradable gift and not a key, you should put instructions for redeeming it into the Message field. Having the winner send you their trade offer URL is easy if you are giving out multiple items. Do not combine redemption message and key fields.

If you are an admin entering a key for another user, please enter their reddit username (prefixed by /u/) or steam username in the 'Donor' field, otherwise you will be credited as the donor. If it was an anonymous gift, please enter Anonymous.

If there is a custom redemption message, the bot will ignore the key fields (both of them); it is assumed that the message contains all information needed to activate the game.
